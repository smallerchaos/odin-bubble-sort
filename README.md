# odin-bubble-sort

Trying to figure out how the flow of things works was pretty difficult, actually. I ended up looking at the Harvard CS50 video but I kind of regret it because I kind of wanted to come up with my own solution first. (They suggested adding a flag to mark if a swap had happened and I totally had an idea to do that but ah well.)

I ended up using loops but I was hoping there was a more elegant way that used built in Ruby methods instead. 

I also used RSpec so I didn't have to manually check to make sure my solution was correct but I ended up having to do that anyway because I wasn't sure if I was bubbling it correctly even if the answer matched.

All in all, I'm quite proud that I was able to figure it out. :)
